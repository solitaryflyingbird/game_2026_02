## §5. event_state JSON 스키마 출력 (lifecycle 시점별)

```
>>> [start_run 직후] event_state = {
  "chain_root_id": "intro_speech",
  "event_id": "intro_speech",
  "kind": "dialogue",
  "line_idx": 0,
  "phase": "awaiting_input"
}

>>> [intro 종료 후] event_state = {}
>>> [intro 종료 후] seen_events = {"intro_speech": 1}

>>> [node 5 진입 직후] event_state = {
  "chain_root_id": "regression_speech",
  "event_id": "regression_speech",
  "kind": "dialogue",
  "line_idx": 0,
  "phase": "awaiting_input"
}

>>> [regression 종료 후] event_state = {}
>>> [regression 종료 후] seen_events = {
  "intro_speech": 1,
  "regression_speech": 1
}

>>> [회귀 직후 (새 internal run 시작 직후)] event_state = {}
>>> [회귀 직후] seen_events = {
  "intro_speech": 1,
  "regression_speech": 1
}
```

(원 출력은 한 줄 JSON, 가독성 위해 줄바꿈.)
