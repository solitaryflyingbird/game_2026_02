## 작업 권장 순서 (톤 일관성 확보 우선)

1. **톤 고정** — 카드 일러스트 + 히로인 포즈 1세트 (스타일 앵커 확정)
2. **몬스터 5종** — 층별 분위기 결정
3. **UI 프레임·아이콘** — 기본 UI 폴리싱
4. **배경** — 공간감 부여
5. **VFX** — 타격감
6. **SFX + 폰트** — 체감 완성도
7. **BGM** — 분위기 마감
8. **코드 측 폴리싱 (히트스톱·쉐이크·Tween)** — 병렬 진행 가능

## 메모

- **열화 팔 카드 12장** 중 열화 변형 6장은 오버레이로 공용 → 실 일러스트는 6장
- **몬스터 포즈** 는 한 장의 베이스 + 살짝 변형으로 처리 가능 (AI 생성 시 일관성 유지 쉬움)
- **팔 분리 렌더링 불채택** (프론트뷰 통합 일러스트)
- **스크린 쉐이크 · 히트스톱 같은 코드 작업**은 에셋과 독립적으로 진행 가능
