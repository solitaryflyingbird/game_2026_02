# 참고 게임 정리 — 적 & 강화 시스템

> 덱빌더 설계 참고용. 본 게임 구조(12장 덱·팔 단위 카드·엄격히 통제된 덱 수정)와 비교할 기준점.
> 공개된 정보 기반이며, 최신 패치·확장팩에 따라 숫자는 달라질 수 있음.

**선정 이유**
- **Slay the Spire (원작)**: 로그라이크 덱빌더의 장르 표준. 베이스라인 비교용
- **Cobalt Core**: 작은 덱(~12~18장) · A/B 분기 업그레이드. 본 게임과 가장 유사한 구조
- **Monster Train**: 다중 업그레이드 슬롯 · 병합/복제 시스템. 단일 카드에 여러 개선이 쌓이는 방식 참고

### 1.2 강화 시스템

- **단일 경로 업그레이드**: 카드 이름 뒤에 `+` 표시. 전부 녹색 톤으로 표기
- **주요 강화 획득처**:
  - **Rest Site — Smith**: 카드 1장 영구 업그레이드 (다른 옵션은 Rest = HP 30% 회복)
  - **유물(Relic)**: 여러 종이 특정 카드·전투 시작 시 카드 업그레이드
    - Whetstone: 시작 시 공격 카드 2장 무작위 업그레이드
    - War Paint: 시작 시 스킬 카드 2장 무작위 업그레이드
    - Tiny House: 1장 무작위 업그레이드 + 카드 3장 선택 획득 + 기타
  - **이벤트**: 다양한 조우 이벤트가 카드 1~N장 업그레이드 보상 제공
- **1회 제한**: 각 카드는 평생 1회 업그레이드. 예외는 **Searing Blow** (무한 업그레이드, 대미지 스케일링)
- **업그레이드 효과 성격**:
  - 수치 증가 (대미지 +N, 블록 +N)
  - 코스트 감소 (2 → 1 등)
  - 특성 추가 (Exhaust 제거, Retain 부여)
  - 카드 종류에 따라 완전히 다른 성격 (어떤 건 수치만, 어떤 건 능력 자체가 바뀜)

**특징**
- 덱 크기 관리가 별개 축: **카드 제거** (상점·이벤트 한정)가 업그레이드만큼 중요
- 업그레이드는 플레이어의 능동적 선택 — 수많은 휴식 노드에서 "쉴까 강화할까" 저울질

### 2.2 강화 시스템

- **A/B 이원 업그레이드**: 모든 카드에 두 개의 고정 강화 경로. 이름 뒤에 `A` 또는 `B`
  - 예: Multi Shot A = 추가 탄약 1발 / Multi Shot B = 쉴드 관통
  - 예: Parry A = Exhaust 제거 / Parry B = Retain 키워드 획득
  - 동일 카드가 다른 덱에서 전혀 다른 역할 수행
- **강화 획득처**:
  - **Repair Yard** (= STS 휴식 노드 유사): **Heal / Upgrade / Remove** 3택 1
  - **Artifact** (= 유물): 특정 카드 자동 강화 / 시작 시 강화 카드 추가 등
  - **조우 이벤트**: 강화·제거·추가 보상이 포함된 이벤트 다수
- **1회 제한**: 각 카드는 A 또는 B 중 하나만 선택. 1회 업그레이드 후 추가 불가
- **스킵 장려**: 카드 보상 스킵 가능 (다른 보상 대체). 덱 얇게 유지가 전략 선택지
- **권장 덱 크기**: 커뮤니티 기준 20장 이하, 25장 초과 비권장. 시작 덱은 15~18장 수준
- **덱 정리 우선**: Basic Shot, Basic Dodge, Basic Defense 같은 기초 카드 제거가 강화보다 우선

**특징**
- 모든 카드에 A/B가 이미 정해져 있음 → 런 중 "새 업그레이드 선택지" 가 아니라 "기존 경로 중 결정"
- 그 결과 카드 설계 단계부터 "이 카드의 A는 공격형·B는 유틸형" 식 디자인 철학 반영
- 덱 크기 조정(제거)과 강화가 **같은 노드**에서 경쟁적 선택지로 나옴 → 매번 고민

## 3. Monster Train

### 3.1 적 (클랜별 기본 구성 + 보스)

Monster Train 은 **링(층)별 적 군집 방어전** 구조. 각 층에 적이 쇄도하며 아래층까지 못 내려가게 막아야 함. 적 카탈로그 총 56종 (+ 19 Boss, 그 중 6 Major Boss).

#### 일반 적 유형 (대표)
- **Imp 계열** — 작고 빠름. Gloating Gremlin, Wee Frosty, Queen's Impling (1코 1/1, 소환 시 정면 10딜) 등
- **Frostbite 부여 유닛** — 상태이상 위주
- **Spikeback** — 반사 대미지 가시 보유
- **Doomed 유닛** — 시간 지나면 자멸. 공격력 높음
- **Stalwart Sentinel, Angelic Lieutenant** — 천사 계열 방어형
- **Spiked / Iceborn / Dreadful** 계열 — 상태이상 분배자
- **Blood Bat, Skelly** — 군집형 약체
- **Incantor, Spiker** — 보조형 유닛 (버프·디버프 부여)

#### Major Boss (6체)
- **Talos, the Architect** — 3링 단독 배치. 공격 전 강화 보너스
- **Archus**
- **Fel, the Judge**
- **Morsel**
- **Herald**
- **Seraph** — 최종 보스 (게임 최후 대결)

각 런의 층마다 등장 적 조합은 난이도(Covenant Rank)와 Pact Shard 설정에 따라 동적 조정됨. 상위 난이도에서 HP·공격력 증폭.

### 3.2 강화 시스템 — 가장 복잡·다층적

#### 업그레이드 슬롯 시스템
- 각 카드에 **업그레이드 2개까지** 장착 가능 (기본)
- 특정 유물 (Pyrestone Housing, Lightstone Casing)로 **3개까지** 확장
- 슬롯이 가득 차면 추가 업그레이드 불가 → 결정 무게 ↑

#### 획득처별 업그레이드 종류
- **Merchant of Steel** (유닛용): 층 사이 상점 노드. 일반 2종(20골드), 희귀 1종(80~100골드)
- **Merchant of Magic** (주문용): 상동
- **Pyre Heart Upgrades**: 덱 수정·전역 효과 부여 (단순 카드 강화 이상)
- **조우 이벤트**: 특정 이벤트에서 특수 강화 획득
- **Champion 업그레이드**: 
  - 런 시작 시 클랜당 1명 챔피언 선택
  - 런 중 특정 노드 (Champion Forge) 에서 **3경로 중 2경로만 선택 가능** (1경로 영구 포기)
  - 각 경로가 다시 분기형이라 조합 폭 큼

#### Duplicate 시스템 (특수)
- 유닛·주문을 **복제**하는 카드·아티팩트 존재
- 복제 시 기존 업그레이드 **함께 계승** (단, 보정치 적용)
  - 본질(Essence) 달린 유닛 복제 시 +15 보정
  - Divine Temple 출신 주문 복제 시 슬롯별 +5/+10 보정
- 즉 고급 강화 카드를 효과적으로 복제하면 덱 파워 스파이크

#### 특수 강화 성격
- **Spikes** — 반격 대미지
- **Multistrike** — 다중 공격
- **Rally / Quick** — 특수 배치·소환 규칙
- **Essence** — 죽을 때 다른 유닛 버프
- **+Attack / +HP** 같은 단순 수치 증강
- 업그레이드 하나하나가 **유닛 전체 역할**을 바꿀 정도의 영향력

**특징**
- **누적 업그레이드**: 한 카드에 여러 개선이 쌓이는 구조 → 단일 카드가 후반엔 완전히 다른 카드로 진화
- **슬롯 한계**로 인한 선택 압박
- **Champion 2/3 분기**가 런마다 "나만의 챔피언" 을 만듬 (매 런 다른 조합)
- **복제·세라프 에센스**가 후반 덱 파워 지수적 증가 허용

## 5. 이 게임에의 시사점

### 5.1 어느 모델이 가장 가까운가?
- **Cobalt Core** 가 구조적으로 가장 근접. 작은 덱 + 강화 중심 + 덱 수정 억제
- **Monster Train** 의 슬롯형 누적 강화는 본 게임 "팔 별 카드 누적 강화" 에 응용 여지
- **Slay the Spire** 는 직접 모델은 아니지만 UX·용어·기대감의 기본값

### 5.2 본 게임 구조의 차별점
| 요소 | 타 게임 | 본 게임 |
|---|---|---|
| 카드 획득 단위 | 카드 단위 추가 | **팔 단위(6장 일괄)** 교체 |
| 카드 상실 단위 | 카드 단위 제거 | **팔 파괴 시 6장 일괄 소실** |
| 강화 단위 | 카드 개별 | 개별 카드 OR 팔 전체 |
| 서사 통합 | 거의 없음 | **각 팔 = 서사적 존재** |

### 5.3 추천 혼합 설계

#### 런 내 강화 — Cobalt Core 식 A/B
- 각 카드에 고정 A/B 경로 사전 설계
- "강펀치 A = 데미지 50 / B = 반동 -10" 같은 대립 분기
- 런 중 휴식/이벤트 노드에서 선택. 1회 한정
- 장점: 카드 설계가 명확하고 깊이 있음

#### 계승 강화 — 회귀 데이터 연동
- 큰 런의 내부 반복 중 누적된 차원종 데이터가 다음 내부 런 초기 히로인에 적용
- 적용 형태:
  - 팔 자체 레벨업 (HP↑, 카드 기본 수치↑)
  - 특수 카드 해금 (기존 카드 교체가 아닌 팔 카드 풀 확장)
  - 특정 조건 강화 (예: "차원종 A 100체 처치 → 해당 종 상대 시 카드 수치 +20%")
- Monster Train 의 Champion 업그레이드 철학 차용 — "이 런에서 한 선택이 영구히 남음"

#### 피하는 것
- **무작위 카드 보상 추가**: 이 게임 덱 구조에 안 맞음. 팔이 곧 카드 풀이라 추가 개념이 어색
- **한 카드에 무한 강화**: Searing Blow 식 스케일링은 서사와 안 맞음 — "그녀의 특정 기술이 무한 강해진다" 는 톤 흐름
- **적 종류 폭발**: 50종 이상 적 제작은 솔로/소규모 팀에 과도. 차원종 5~10종 + 변종·보스로 충분

### 5.4 적 풀의 현실적 규모
- **Cobalt Core** 수준(~25종)이 인디 기준 적정선
- 본 게임 목표 제안:
  - 일반 차원종 8~12종
  - 엘리트 변종 3~5종
  - 주요 거점 보스 3~5종
  - 최종 둥지 보스 1체
  - 합계 **15~20종** — 서사의 응집력·제작 부담·플레이 다양성의 균형점

## 6. 추가 조사가 필요한 영역

- Slay the Spire 의 **이벤트 카드 강화** 세부 (특정 이벤트에서 무료 강화 vs 대가 요구)
- Cobalt Core 의 **아티팩트 강화 효과** 전체 목록
- Monster Train 의 **Pyre Heart 업그레이드** 전체 풀
- 참고군 외의 소규모 덱빌더: **Wildfrost** (유닛 기반), **Dicey Dungeons** (아이템 기반), **Griftlands** (이중 덱), **Inscryption** (내러티브 중심)
- 2024~2026 신작 흐름 (장르 포화 상황에서 차별점 전략 연구)
