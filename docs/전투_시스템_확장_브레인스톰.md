> 두서없는 대화에서 도출된 핵심 메커닉·서사 결정 사항.
> 기존 `영원회귀_네러티브.md` 의 §9.4 (사랑의 비대칭) · §10 (장르) 와 연동.

## 1. 콤보 시스템 (크리티컬 히트)

### 1.1 기본 규칙
- 각 적에 **약점 수** (1~9 중 하나) 배정
- 한 전투 동안 사용한 **누적 마나** 를 **mod 10** 으로 추적
- 누적 마나의 1의 자리 = 약점 수 되는 순간의 카드 → **크리티컬**
- 크리티컬 = **데미지 ×2** + 전용 컷인 아니메

### 1.2 예시
- 적 약점 7
- 1턴: 4마나 (누적 4)
- 2턴: 1마나 (누적 5) → 2마나 (누적 7) → **크리**
- 이후 계속 누적 → 9+2 = 11 → mod 10 = 1 → 약점 1 적 공략 가능

### 1.3 원본 팔 독점
- **원본 팔 카드만** 누적 + 크리 기여
- **열화 팔** 카드: 크리 불가 + 누적에 영향 없음 (존재하지 않는 것처럼)
- **특수 팔** 카드 (장군 토벌 보상): 동일. 크리 불가 + 누적 무관

### 1.4 서사·전술 의의
- **원본 팔의 수학적 리듬** = 그녀의 설계된 온전함 (amor fati 시각화)
- 원본 파괴 = 수치 너프 + **콤보 상실** 이중 페널티
- 양쪽 원본 다 잃으면 **콤보 완전 상실** → 전투가 기계적 반복으로 전락
- 노말 엔딩 직전 상태 시각적 표현: *"그녀의 춤이 멈춤"*

### 1.5 UI 요구사항
- HUD 에 **누적 마나 (1의 자리)** 큰 숫자로 상시 표시
- 적 HP 바 근처에 **약점 수** 표기
- 손패 중 **크리 가능 카드 하이라이트** (옵션 ON/OFF)
- 크리 발생 시 컷인·힛스톱·2배 수치 팝업

### 1.6 확장 여지 (후순위)
- 보스 복수 약점 (예: 약점 3 AND 7)
- 약점 변동형 엘리트 (매 N턴 약점 바뀜)
- 약점 면역 적 (희귀)
- 누적 마나 조작 유틸 카드

## 2. 4 장군 구조

### 2.1 핵심 규칙
- 맵에 **4 장군** 배치 (고정 노드)
- **모두 토벌해야** 진 엔딩 경로 진입 가능
- 미완료 시: 쉘터 도달해도 적 수 과다 → 주인공 생존 위해 **자폭 강제** → 노말 엔딩 루트

### 2.2 장군별 특수 팔 드랍
- 토벌 시 해당 장군의 **특수 팔** 획득
- 특수 팔 = **메가맨식 위크니스 체인**: 한 장군의 팔이 다른 장군의 약점
- 플레이어의 공략 **순서 전략** 창출

### 2.3 서사적 의미
- *"차원종의 잔재를 무기 삼는다"* — 적을 흡수한 그녀
- 진 엔딩 도달 과정 = **그녀가 점점 자기 자신이 아니게 되는 대가**
- 결전 직전의 그녀 외형 = **원본 팔 + 흡수된 차원종 조각들의 혼합체**

## 3. 특수 팔 시스템

### 3.1 공통 특성
- 원본보다 **기본 성능 낮음** (~70~80%)
- **콤보 불가** + **누적 마나 무관**
- 하나 이상의 **특화 성능** 보유 → 특정 적·상황에 압도적

### 3.2 설계 예시 (4종 제안)

#### 극딜 검 팔 (장군 A 드랍)
- 공격 카드 위력 ×1.5 (원본 대비)
- 공격당 **자체 HP 5 소모** (누적)
- 단기전 특화. 장기전 자멸
- 상성: 재생형·고HP 단일 보스

#### 화염방사 팔 (장군 B 드랍)
- 공격 카드가 **모든 적 동시 타격** (각 60~70% 데미지)
- 단일 적엔 효율 낮음
- 상성: 군집·소환형 장군

#### 관통 팔 (장군 C 드랍 — 제안)
- 적 블록·방어 **완전 무시**
- 단일 데미지 원본 수준
- 상성: 중장갑·방패 스택형

#### 빙결 팔 (장군 D 드랍 — 제안)
- 공격 + 적 1턴 행동 무력화
- 공격력 ~65%
- 상성: 타임드 버스트·스케일러

### 3.3 장착 규칙
- L / R 슬롯 2개 (기존 구조)
- 쉘터·휴식 노드·메뉴에서 언제든 교체 가능
- **전투 중 교체 불가** (시스템·UI 양쪽에서 가드 — 이미 구현됨)
- 한 번 획득한 특수 팔은 **큰 런 내 영구 보유**. 노말 엔딩 리셋 시 초기화

### 3.4 전술 조합 예시
- 원본 L + 원본 R: 콤보 최대 · 범용
- 원본 L + 극딜 R: 콤보 절반 + 단일 burst 옵션
- 화염 L + 관통 R: 콤보 0 + 극단 특화. 장군전용

### 3.5 특수 팔 HP·파괴
- 특수 팔도 HP 있음. 원본보다 낮거나 비슷
- 파괴 시 이번 큰 런 **영구 상실** (회복 불가)
- 다음 큰 런 시작하면 다시 장군 토벌로 획득

### 3.6 특수 팔 강화
- A/B 분기 업그레이드 적용 (원본과 동일)
- 콤보 없는 대신 강화는 가능

## 4. 경험치 시스템

### 4.1 획득
- 적 격파: 기본 XP
- 콤보 크리 발동: 보너스 XP
- 장군 격파: 대량 XP
- 열화 팔로만 싸운 전투: XP 페널티 (원본 보전 유도)

### 4.2 성장
- 레벨별 **콤보 관련 강화** 누적
- 1~2 레벨 증가는 **체감 미미** (의도)
- 그라인드 유도 방지 — 자연스럽게 쌓이는 부산물

### 4.3 레벨 보상 예시
| Lv | 효과 |
|---|---|
| 1 | 크리 ×2 (기본) |
| 2~3 | 크리 +10%씩 |
| 4 | 약점 ±1 범위 확장 |
| 6 | 크리 시 양팔 블록 +10 |
| 8 | 크리 연쇄 (다음 카드 자동 크리) |
| 10 | 약점 ±2 범위 확장 |
| 16 | 홀·짝 전체 크리 |
| 20+ | 전 공격 크리 (결전 영역) |

### 4.4 내부 런 단위 도달 범위
- 일반 내부 런 완주 시 Lv 3~5 수준
- 결전 진입 시 **누적된 모든 내부 런 XP 통합** → Lv 15~25 이상 가능
- 원본 팔 유지 시 XP 획득 효율 ↑. 잃으면 쌓이는 속도 감소

### 4.5 결전 시 추가 조건
- 원본 팔 양쪽 **쉘터에서 수리·복원** (파괴 상태였어도)
- 누적 회차 연구 데이터로 복원 기술 완성
- 초반 큰 런에선 복원 기술 부족 → 결전 불가 · 후반 큰 런에서 가능

### 4.6 UI
- 내부 런 중: 작은 숫자로 현재 Lv 은근히 표시
- 결전 직전: 몽타주 후 **"통합 Lv N"** 숫자 점프 연출

## 5. 최종결전 디자인 철학

### 5.1 핵심 선언
**결전은 시험이 아니라 보상**. 기계적 난이도 거의 없음. 도달 자체가 증명.

### 5.2 이유
- 내부 런들 = 실질 시험 (전술·판단·반복)
- 결전 = **감정적 피크 · 서사적 완결**
- 또 시험 주면 **감정선 꺾임**
- *"너는 이미 증명했다. 이제 즐겨라"*

### 5.3 난이도 설계
- 히로인 압도적 스탯 (누적 Lv + 원본 양팔 + 모든 기억)
- 결전 보스 HP 대비 데미지 극도로 높게 세팅
- 명백한 태업 (턴 그냥 넘김·일부러 아군 공격 등) 에만 패배 가능
- **연출로 압도감 전달**: BGM·컷씬·스프라이트 변화가 주역

### 5.4 참고 사례
- Persona 5 후반 보스: 오버스펙 압도 의도
- NieR Automata E 루트: 기술적 난이도 낮음, 감정이 본질
- Outer Wilds 엔딩: 전투 없음. 도달이 보상

### 5.5 이스터에그
- **"당신 일부러 졌죠"**: 결전에서 일부러 지면 특수 대사·분기
  - 히로인: *"…당신, 일부러 지신 거죠."*
  - *"알아요. 이렇게 끝내고 싶지 않으셨군요."*
  - *"저도 그래요. 한 번 더 가볼까요?"*
  - → 재시도 허용 혹은 특수 엔딩
- 반복 태업 시 더 짓궂은 대사 가능

## 6. 히로인의 감정 아크

### 6.1 기억 계승 전 (대부분의 게임 시간)
- **"나 따위가 감히..."** 의 벽
- 전투 유닛 자기 규정. 주인공의 사랑에 답할 자격 없다고 여김
- 사랑을 **임무 수행** 으로 대체. 정서적 거리감 유지
- 선행 불가. 주인공의 감정 표현도 조용히 받음

### 6.2 기억 계승 순간 (결전 진입 직전)
- 0회차부터의 모든 쉘터 도달·자폭 기억 전송
- 주인공이 **N회 반복하며 얼마나 버텼는지** 직접 목격
- 자기 평가 전복: *"내가 이렇게까지 사랑받았구나"*
- 동시에 자기 힘도 인식: *"이번엔 이길 수 있다 = 돌아올 수 있다"*
- 벽 붕괴

### 6.3 기억 계승 후
- **적극적 감정 표현** 으로 전환
- 선제 키스 (주인공의 망설임을 깨뜨림)
- *"갔다와서 할게요"* (엔드 오브 에반게리온 Misato 오마주. 이번엔 **실현됨**)
- 결전 후 돌아와서 약속 완성

### 6.4 결전 구도
- **그녀 혼자** 적진 진입 (주인공 쉘터 대기)
- 이유: 결전 지역 위험도 극심. 주인공 생존 위해 격리
- 서사 의미:
  - 역할 역전: 그동안 그가 지켜봤던 것처럼 이번엔 **그가 대기**
  - 그도 처음으로 *"닿을 수 없음의 무력감"* 체험 (그녀가 평생 느낀 것)
  - 주인공의 amor fati 시험 순간

### 6.5 플레이어 시점 (결정 필요)
- **옵션 A**: 주인공 시점 유지 (쉘터 대기, 전투는 컷씬)
- **옵션 B**: 히로인 시점 전환 (처음으로 그녀 조작)
- **옵션 C**: 하이브리드 (전투는 히로인, 컷씬은 주인공)
- **임시 추천**: C

## 7. 엔딩 톤 (진 엔딩)

### 7.1 기본 방향
게임 내내 무거운 톤 → 엔딩에서 **의도적 경량화**. 평화·일상·작은 유머.

### 7.2 구성 시퀀스
1. 결전 승리 + 스톤 파괴
2. 그녀 귀환 + 약속 실현 장면
3. 시간 경과 (몇 달~몇 년)
4. 인류 재건 몽타주 짧게
5. 공적 인정식 (그녀가 조용히 상 받음)
6. 일상 진입
7. **백화점 메이드로봇 씬** (캐릭터 비트)
8. 시간 더 경과
9. **풀밭 정원 무릎배게 낮잠 씬** (최종 비주얼)
10. 엔딩 크레딧 + 일상 몽타주

### 7.3 백화점 씬의 의미
- 그녀가 **무언가를 원함** 자체가 엔딩의 증거
- 이전엔 욕망 없음 (욕망은 주인공 차지)
- 메이드 로봇 바디 = 전투 외 자기 가능성 탐색. 가정적·여성적 자아
- 주인공의 반쯤 당황한 반응으로 유머 비트 성립

### 7.4 정원 씬의 의미
- 클래식 평화 상징 (애니 전통의 정지 이미지)
- 이 게임 맥락: **그가 더 이상 지시할 필요 없고, 그녀가 더 이상 싸울 필요 없다**
- 역할·관계 둘 다 해소 상태. 그냥 존재만

### 7.5 연출 디테일
- BGM: 피아노 솔로 혹은 무음 + 자연음
- 카메라: 롱 테이크·슬로우 줌아웃
- 대사: 최소. 독백 한 줄 있거나 없음
- 예: *"…또 반복될까요, 이런 순간도."* / *"…반복되길. 영원히."*

### 7.6 크레딧 구성
- 엔딩 음악 유지
- 배경에 **일상 몽타주 컷들 연속 재생** (식사·산책·독서·별 보기·비 오는 창 등)
- 각 컷 5~10초. 대사 없음
- *"amor fati 의 완성 = 평범한 일상을 사랑하는 것"* 의 시각 표명

### 7.7 포스트 크레딧 (선택)
- 짧은 한 컷
- 파괴된 스톤 서랍 안. 주인공이 닫음
- 혹은 밤하늘을 바라보는 그녀 (자연 회귀 암시)

## 8. 노말 vs 진 엔딩 대비 강화표

| 요소 | 노말 | 진 |
|---|---|---|
| 반격 | 포기 | 완수 |
| 세계 | 둥지 여전·숨어 삶 | 둥지 제거·재건 |
| 주인공 | 자연사 | 생존 |
| 히로인 | 홀로 남음 | 함께 살아감 |
| 사랑 비대칭 | 끝내 미해소 | 결전 순간 해소 |
| 키스 | 없음 (벽 유지) | 그녀가 선제 |
| 4 장군 | 미토벌 | 완료 |
| 원본 팔 | 복원 불가 | 결전 위해 복원됨 |
| 콤보 | 후반 상실 | 결전에서 완전 개화 |
| 최종 비주얼 | 쉘터 실내·임종 | 정원·낮잠 |
| 주인공 마지막 | *"너는 너답게 있었어"* | *"…"* (말 잃음) |
| 히로인 마지막 | *"멋진 일이었어요, 정말"* | *"다녀왔어요"* |
