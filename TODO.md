# LAST MISSION 최후의 임무 - 개발 진행 현황

## 프로젝트 개요

- **장르:** 로그라이크 덱빌딩 + 주사위 퍼즐
- **컨셉:** 폐기 직전 로봇의 마지막 임무
- **엔진:** Godot 4.6 / 해상도 1280x720
- **스프라이트 규격:** 48x48px 픽셀아트

## 완료된 작업

### 핵심 전투 로직
- [x] 주사위 덱 시스템 (ATK, BLK, BST, HEL x 등급 1~4, 등급별 7장)
